Chapter 3  Photo Browsing

This Chapter introduces the way to browse photos by using GPS photo browser.

 Function and features

1. Support browsing of JPG,PNG and TIFF format photos;

2. Support photo rotation, zoom in/ out, auto play.

Please copy your photo file into any folder of the SD card before browsing photo. 

Interface and operation

Click    "Photo browse" icon on main interface of system to open photo browser as shown in Picture 5-1.

 

Close

Click to close photo browser;

Zoom in

Zoom in current photo proportionally

Zoom out

Zoom out current photo proportionally;

Rotate photo

Click to rotate current photo clockwise 90 degree;

Auto play

Full screen circle display all photos of a folder from current photo;

Previous

Browse the previous photo of the same folder;

Next

Browse the next photo of the same folder;

Folder

Click to switch photo folder interface to choose preferred photo(see Picture 5-2);

File name & format

Display current photo name and its photo format.
